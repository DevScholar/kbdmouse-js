# KBDMouseJS

⚠️ This project is still in pre-alpha stage, and API is subject to change.

## Introduction

This is a virtual keyboard library that supports keyboard events aim to be used
in web-based emulators or legacy web pages that depend on keyboard events on
mobile devices.

## Compatibility

- [x] v86 (old version)
- [x] em-dosbox

## Usage

```html
<!-- Include the library and stylesheet in dist folder, then -->
<virtual-keyboard></virtual-keyboard>
<!-- That's all.-->
```
